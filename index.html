Downloads

Grid criado para usar internamente na Agência Monk, com apenas 7KB

Link

Você pode usar o monk-grid direito do projeto no git.

<link rel="stylesheet" type="text/css" href="https://agenciamonk.github.io/monk-grid/flexalign.css">

ou clonar ele

$ git clone https://github.com/AgenciaMonk/grid.git

Instalação com bower

Você tambem pode fazer o download via bower.

$ bower install monk-grid

nao se esqueça de linkar...

<link rel="stylesheet" type="text/css" href="bower_components/monk-grid/flexalign.css">

Instalação com npm

Tambem oferecemos a opçao de download via npm.

$ npm install monk-grid

nao se esqueça de linkar...

<link rel="stylesheet" type="text/css" href="node_modules/monk-grid/flexalign.css">

Instalação com Yarn

Você tambem pode fazer o download via bower.

$ yarn add monk-grid

nao se esqueça de linkar...

<link rel="stylesheet" type="text/css" href="node_modules/monk-grid/flexalign.css">
