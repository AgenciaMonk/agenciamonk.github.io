#CONTAINER

Toda div pai deverá ter a classe CONTAINER.

#ROW ou COLUMN

Utilizando ROW a posição sera Horizontal, COLUMN a posição sera Vertical.

Ler mais

#WRAP ou NOWRAP

WRAP significa que os itens vão quebrar quando não tiverem mais espaço. Para o WRAP funcionar é obrigatório os itens terem a classe BASIS(BASIS1 A BASIS12).

NOWRAP já é default, ela impede que os itens quebrem quando não tiver espaço.
