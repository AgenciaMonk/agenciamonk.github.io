Aprenda a usar o Framework


O grid é baseado em CONTAINER e ITEM.

#CONTAINER

Toda div pai deverá ter a classe CONTAINER.

<div class="container">
    <div class="item">
         ...
    </div>
  </div>

Classes para o container

  • CONTAINER*
  • ROW OU COLUMN
  • WRAP OU NOWRAP
  • TOP-LEFT
  • TOP-CENTER
  • TOP-BOTTOM
  • CENTER-LEFT
  • CENTER-CENTER
  • CENTER-BOTTOM
  • BOTTOM-LEFT
  • BOTTOM-CENTER
  • BOTTOM-BOTTOM
  • space-between
  • space-between-top
  • space-between-center
  • space-between-bottom
  • space-between-stretch
  • space-between-baseline
  • space-around
  • space-around-top
  • space-around-center
  • space-around-bottom
  • space-around-stretch
  • space-around-baseline

ex: 
<div class="container wrap center-center">
    <div class="item">
         ...
    </div>
  </div>
