Buttons

Simple button fixed class: btn


First add the class btn, for the default settings, it has a default height of 40px height and font-size of 12px.

<button class="btn">Enviar</button>

<button class="btn btn-primary">Enviar</button>

<button class="btn btn-sucess">Complete</button>

<button class="btn btn-cancel">Cancel</button>

Button small add class: btn-small

Se preferir ele menor(height:25px font-size:10px), adicione a classa BTN-SMALL,

<button class="btn btn-small">Enviar</button>

<button class="btn btn-primary btn-small">Enviar</button>

<button class="btn btn-sucess btn-small">Complete</button>

<button class="btn btn-cancel btn-small">Cancel</button>

Button big add class: btn-big

<button class="btn btn-big">Enviar</button>

<button class="btn btn-primary btn-big">Enviar</button>

<button class="btn btn-sucess btn-big">Complete</button>

<button class="btn btn-cancel btn-big">Cancel</button>

Simple button radius(4px) add class: btn-radius

<button class="btn btn-rounded">Enviar</button>

<button class="btn btn-primary btn-rounded">Enviar</button>

<button class="btn btn-sucess btn-rounded">Complete</button>

<button class="btn btn-cancel btn-rounded">Cancel</button>

Simple button rounded add class: btn-rounded

<button class="btn btn-radius">Enviar</button>

<button class="btn btn-primary btn-radius">Enviar</button>

<button class="btn btn-sucess btn-radius">Complete</button>

<button class="btn btn-cancel btn-radius">Cancel</button>

Button full add class: btn-full

<button class="btn btn-full">Enviar</button>

<button class="btn btn-primary btn-radius btn-full">Enviar</button>

<button class="btn btn-sucess btn-rounded btn-full">Complete</button>

<button class="btn btn-cancel btn-small btn-full">Cancel</button>

<button class="btn btn-cancel btn-rounded btn-big btn-full">Cancel</button>
